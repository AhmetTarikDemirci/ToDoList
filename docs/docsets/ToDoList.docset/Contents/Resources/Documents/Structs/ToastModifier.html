ToastModifier

struct ToastModifier : ViewModifier

İçerik görünümünün üzerine bir bildirim (toast) mesajı yerleştiren bir görünüm değiştirici.

Bu değiştirici, belirli bir süre boyunca ekranın alt kısmında bir bildirim mesajı göstermenizi sağlar. Bildirim mesajı manuel olarak veya belirli bir süre sonra otomatik olarak kapatılabilir.

Örnek kullanım:

struct ContentView: View {
    @State private var toast: Toast?

    var body: some View {
        VStack {
            Button("Bildirim Göster") {
                toast = Toast(
                    style: .success,
                    message: "Bu bir bildirim mesajıdır!",
                    width: 300,
                    duration: 2.0
                )
            }
        }
        .modifier(ToastModifier(toast: $toast))
    }
}
